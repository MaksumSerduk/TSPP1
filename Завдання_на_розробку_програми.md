
## Завдання на створення програми 
Розробити програму, яка читає дані із файлу БД MySQL та дозволяє користувачеві:
- вводити нові записи
- редагувати існуючі записи
- робити обробку даних за заданими параметрами

### Структура бази даних

Файл БД MySQL містить довідки про **асортимент дитячих іграшок у магазині**.

Структура запису:

| Поле | Опис |
|:---|:---|
| Назва іграшки | найменування товару |
| Ціна | вартість одиниці товару |
| Кількість | наявна кількість на складі |
| Вікова межа (від) | мінімальний рекомендований вік дитини |
| Вікова межа (до) | максимальний рекомендований вік дитини |

**Кількість записів:** довільна, але не менше **10** записів.

### Функціональні вимоги

Програма повинна видавати у файл **MS Word** таку інформацію:

1. **Список іграшок (назви)**, що підходять дітям у віковому діапазоні від X до Y років.
2. **Інформацію про найдешевшу іграшку** серед обраних за віковим критерієм.

> Значення **X** (мінімальний вік) та **Y** (максимальний вік) вводяться користувачем у відповідну форму програми.


### Ролі користувачів

**Користувач (робітник магазину)** може:
- переглядати весь асортимент у головній формі
- додавати записи про нові іграшки
- редагувати дані про наявні товари
- виконувати пошук/фільтрацію за віковими межами (X, Y)
- зберігати результати пошуку та інформацію про найдешевший товар у файл MS Word
